    Orders¶

    The order management API lets you place a new order, cancel or modify the pending order, retrieve the order status, trade status, order book & tradebook

    Type Apis Details
    POST order/execute Place a new order
    POST order/modify Modify a pending order
    POST order/cancel Cancel a pending order
    POST order/getmargin get order margin
    POST orderinfo/orderbook Retrieve the list of all orders for the day
    POST orderinfo/tradebook Retrieve the list of all trades for the day
    POST orderinfo/history get order history

    Modify Order¶

    Request Structure

    {
    "userId": "<USER_ID>",
    "exchange": "NSE",
    "tradingSymbol": "ZOMATO-EQ",
    "orderNo": "23011900000017",
    "qty":"2",
    "ret": "DAY",
    "priceType": "MKT",
    "tranType": "B",
    "product": "I",
    "price": "0",
    "triggerPrice": "",
    "disclosedQty": "",
    "MktProtection": "",
    "bookProfitPrice": "",
    "bookLossPrice": "",
    "trailingPrice": ""
}
    Input parameters

    Field Type Description
    userId string User specific identification generated by nidhi
    exchange string Order specific identification generated by nidhi
    tradingSymbol string Order Type
    orderNo string In case of BO & CO, which leg is modified `
    qty int Number of shares for the order
    ret float Price at which order is placed
    priceType int Number of shares visible (if opting keep >30% of quantity)
    tranType float Price at which the order is triggered, in case of SL-M & SL-L
    product string Validity of Order
    price string Validity of Order
    triggerPrice string Validity of Order
    disclosedQty string Validity of Order
    MktProtection string Validity of Order
    bookProfitPrice string Validity of Order
    bookLossPrice string Validity of Order
    trailingPrice string Validity of Order

    Response Structure

    {
    "status": "Ok",
    "message": "Success",
    "result": [
        {
            "requestTime": "17:52:09 22-02-2023",
            "orderNo": "23022200000028"
        }
    ]
}
    Parameters

    Field Type Description
    requestTime string Order specific identification generated by nidhi
    orderNo string Last updated status of the order
